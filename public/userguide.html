Doctor Schedule

Manage doctor schedules, shift assignments, and availability for appointments and consultations.

Key Features:
  • Add / Edit / Delete Doctor Appointment Schedule
  • Mandatory Fields: Marked with a red *
  • Schedule Setup:
    • Start Time / End Time: Use 24-hour format (e.g., 08:00 to 13:00)
    • Average Appointment Duration (minutes): Defines how long a doctor spends per appointment. Used in Serial Type
    • Serial Type:
      • Sequential: Appointments are slotted one after another
      • Timestamp: Time-based slots are created by dividing the total available duration by the average appointment length
      • Example: If schedule is from 09:00 to 12:00 (3 hours = 180 minutes) and average appointment duration is 30 minutes → 6 slots (09:00–09:30, 09:30–10:00 … 11:30–12:00)

Clinical Leaderboard

Performance tracking system that displays clinical staff activities and rankings based on actual data from the system.

How Clinical Leaderboard Works:

The Clinical Leaderboard automatically tracks and ranks clinical staff based on their documented activities within the system. It provides real-time performance metrics and detailed activity breakdowns.

Eligible Providers:

The leaderboard includes staff with the following roles: Doctor, Nurse, Admin, Super Admin, and Physiotherapist.

Metrics Tracked:
  • Patients Seen: Count of unique patients interacted with across all activities
  • Appointments Conducted: Total number of patient appointments completed
  • Clinical Reports Submitted: Combined total of all clinical documentation including:
    • Therapy Reports (by doctor_id)
    • Prescriptions (by doctor_id)
    • Lab Requests (by doctor_id)
    • Radiology Requests (by doctor_id)
    • Nursing Cardexes (by doctor_id)
    • Drug Orders (by doctor_id)
Filtering Options:
  • Date Range: Filter by start and end dates (defaults to last 30 days)
  • Provider: Filter by specific provider or view all providers
  • Automatic Updates: Rankings update automatically based on filtered criteria
Detailed View:

Click "View Details" for any provider to see comprehensive breakdown including:

  • Appointments Conducted: Date, time, patient, and problem description
  • Therapy Reports: Date, type, patient, and notes
  • Prescriptions: Date, patient, medication, and dosage
  • Lab Requests: Request date, patient, test type, and status
  • Radiology Requests: Request date, patient, procedure, and status
  • Nursing Cardexes: Date, patient, shift, and notes
  • Drug Orders: Date, patient, medication, and status
Ranking System:
  • Primary Sort: Ranked by "Clinical Reports Submitted" in descending order
  • Display Threshold: Only shows providers with at least one activity (reports, patients seen, or appointments)
  • Real-time Data: All data is pulled directly from the system tables in real-time
